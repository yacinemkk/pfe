Diapositive 9

Démarche Choisie

  • 5/26
  • 1. Dataset : IoT IPFIX Home (2023) : Données réelles issues de 12 foyers (Japon).
  • Durée de collecte : 47 jours.
  • Diversité : 24 types de dispositifs IoT variés (caméras, capteurs, domotique, multimédia).
  • 2. Dataset : IPFIX Records (2022):Banc d'essai résidentiel avec 26 objets IoT sur 3 mois.
  • Volume massifs : Plus de 9 millions d'enregistrements basés sur 28 attributs de flux.
  • Pipeline d'extraction : Trafic brut (PCAP) ➔ Fichiers Binaires (via l'outil YAF)
  • ➔ Format JSON (via Super Mediator).
